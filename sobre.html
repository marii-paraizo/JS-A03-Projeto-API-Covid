Projeto: Gráficos de Dados de Casos de COVID-19

Descrição do Tema

Este projeto apresenta gráficos que visualizam dados de casos de COVID-19 tanto no Brasil quanto em nível mundial. Utilizando informações atualizadas, os gráficos permitem uma análise clara da evolução da pandemia, facilitando a compreensão das tendências e do impacto em diferentes regiões e países.

Objetivo

O principal objetivo deste projeto é fornecer uma visualização intuitiva e acessível dos dados relacionados à COVID-19, permitindo que os usuários acompanhem a situação da pandemia em tempo real e compreendam melhor as informações epidemiológicas tanto localmente quanto globalmente.

Créditos sobre a Origem dos Dados

Os dados utilizados para a criação dos gráficos são provenientes de duas fontes principais: API COVID-19 do Brasil, que fornece informações sobre casos confirmados, mortes e recuperações no Brasil. API de Dados Mundiais de COVID-19, que disponibiliza dados atualizados sobre a situação da pandemia em diferentes países. Para mais detalhes e informações, consulte a documentação das APIs mencionadas. Este projeto visa contribuir para a disseminação de informações cruciais sobre a pandemia, promovendo a conscientização e a educação da população em escala local e global.
